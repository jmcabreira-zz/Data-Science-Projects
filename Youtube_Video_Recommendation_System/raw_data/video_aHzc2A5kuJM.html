Published on Dec 29, 2019

South China University of Technology published a research paper about facial beauty prediction. They also open-sourced the data set (https://arxiv.org/pdf/1801.06345.pdf ).

So, a beauty score prediction application is built based on that study. Besides, accuracy metrics is much more successful than the reference study. Then, the built machine learning model is run on IMDb database (https://data.vision.ee.ethz.ch/cvl/rr... ). Almost 50K actress exist in the the database.

These are the 25 most beautiful actresses in IMDb data set in the view of an AI system. BTW, photos illustrated in this video is taken from the IMDb data set that the AI system scores.
