Published on Sep 15, 2015

There are many cloud services that support automated code deployment from GitHub. In this video, we take a look at connecting a GitHub repository to a Microsoft Azure instance. When working with continuously delivered projects like web apps, this connection can be especially helpful for pushing out things like critical bug fixes; but it also removes barriers that might be keeping new features in a holding pattern on stale branches.
