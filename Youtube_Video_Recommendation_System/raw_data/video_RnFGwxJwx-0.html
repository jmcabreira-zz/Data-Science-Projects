Machine learning is changing the world that we live in. Top companies such as Facebook, Google, Microsoft and Amazon are looking for machine learning engineers and the average salary of a machine learning engineer is around 120k$ dollars.
So, keeping the importance of machine learning in mind, we have come up with this session, so that you can get started in the field of machine learning.

These are the topics covered in this session:

-Introduction to Python and Anaconda
-Introduction to Pandas and Data Manipulation
-Introduction to Numpy and Numerical Computing
-Data Visualization
-Statistics vs Machine Learning
-Types of Statistics
-Understanding Data
-Reinforcement Learning
-Q-Learning Algorithm
-Case Study - Smart Taxi
