Machine Learning Operations (MLOps) with Microsoft Azure by Eumar Assis

Intellinsight LLC in collaboration with the Microsoft Data and AI - South Florida user group presents Machine Learning Operations (MLOps) with Azure by Eumar Assis from Microsoft.

The Machine Learning Operations (MLOps) with Microsoft Azure is about operationalizing ML workflows with Azure Machine Learning. MLOps is based on DevOps principles and practices that increase the efficiency of workflows. MLOps applies DevOps principals to the data science process, bringing continuous integration, delivery, and deployment. MLOps enables faster experimentation and development of models, faster deployment of models into production, and quality assurance. In this session, we will uncover how Microsoft Azure Machine Learning Service is used to build MLOps practices in the Cloud.
