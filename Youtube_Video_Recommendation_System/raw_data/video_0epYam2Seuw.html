There are many different apps you can deploy using Intune. This Nugget demonstrates this and discusses important things to think about.

This video is from Microsoft Windows 10: Manage Apps and Data, which maps to Managing Modern Desktops (MD-101).

Join Anthony Sequeira as he teaches you how to manage Windows 10 apps and data. Learn how to assign apps to groups, how to secure data with Intune, how to enable sideloading of apps into images, how to deploy apps with Intune and Microsoft Store for Business, and how to implement and manage Mobile Application Management (MAM) policies. Gain an understanding of O365 ProPlus, Windows Information Protection, IE Enterprise mode, and Azure Information Protection templates.
