Published on Feb 14, 2020

Prof. Matthew Lanham talks with NVIDIA ambassador & trainer Dwayne Paschall about the NVIDIA Deep Learning Institute (https://www.nvidia.com/en-us/deep-lea...), and how GPUs and Natural Language Processing (NLP) are being used today in business. Dwayne held a one-day workshop at Purdue for MS BAIM students on the Fundamentals of Deep Learning for NLP. This workshop provided hands-on experience using classical approaches to convert text to machine-understandable data representations, implementation and properties of distributed representations (embeddings), and methods to train machine translators from one language to another using Python TensorFlow and Keras libraries on top of NVIDIA GPUs.
